Vytvořit nové QR

Tento formulář vytvoří nový záznam v codes a zároveň automaticky založí první obsahovou verzi v code_versions jako verzi 1.

Tento název se uloží do codes.title i do první verze v code_versions.title.
Nech prázdné a aplikace vygeneruje bezpečné ID automaticky. Fyzický QR kód pak bude směřovat na index.html?code=....
Ukládá se jako první verze obsahu. Při budoucích úpravách se má vytvářet nová verze, ne přepis původní.
QR bylo úspěšně vytvořeno.
Veřejný odkaz:
Admin detail:

Co se uloží

1) Tabulka codes Vznikne hlavní QR záznam: id, title, owner_id, active = true a kvůli kompatibilitě i schema.
2) Tabulka code_versions Okamžitě vznikne první aktivní verze: version_number = 1, stejné title, stejné schema, is_active = true.
3) Konzistence verzování Veřejný formulář pak čte aktivní verzi z code_versions. Staré odpovědi zůstávají navázané na staré verze.

Ukázkové schema

[
  {
    "type": "text",
    "name": "jmeno",
    "label": "Jméno",
    "required": true
  },
  {
    "type": "textarea",
    "name": "poznamka",
    "label": "Poznámka",
    "required": false
  },
  {
    "type": "radio",
    "name": "stav",
    "label": "Stav",
    "required": true,
    "options": ["OK", "Závada"]
  }
]
Poznámka: pokud chceš později schema změnit, fyzický QR kód zůstává stejný. Mění se jen obsah přes novou verzi.
